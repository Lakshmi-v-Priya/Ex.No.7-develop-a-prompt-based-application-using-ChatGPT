# Experiment No: 7  
## Develop a Prompt-Based Application Using ChatGPT

**Date:**  
**Register No:**  

---

## Aim
To develop a prompt-based application using ChatGPT and demonstrate how to create a prompt-based application to organize daily tasks, showing the progression from simple to more advanced prompt designs and their corresponding outputs.

---

## AI Tools Required
- ChatGPT (Large Language Model)
- Web Browser
- Markdown Editor / Text Editor
- Optional: Python or Command Line Interface for simulation

---

## Explanation
A prompt-based application uses prompts to interact with a Large Language Model (LLM) such as ChatGPT. By designing well-structured prompts, users can create intelligent assistants capable of managing tasks, answering queries, and generating suggestions.

In this experiment, a **Personal Productivity Assistant** is designed. The assistant helps users manage daily activities, schedule reminders, provide wellness tips, and answer general questions using natural language interaction. The prompts can be refined to make the assistant more adaptive and useful.

---

## Prompt
Design a personal productivity assistant that can help manage daily tasks, schedule reminders, suggest wellness tips, and answer general queries. The assistant should interact using natural language and be adaptable to the user’s changing preferences over time.

---

## Procedure

1. **Define the Core Requirements**
   - Identify the features required in a personal productivity assistant such as task management, reminders, wellness tips, and query answering.

2. **Design Prompts for Each Feature**
   - Create prompts that instruct the LLM to perform specific tasks.
   - Example prompts:
     - “Add a new task: Complete assignment by 5 PM.”
     - “Remind me to attend the meeting tomorrow at 10 AM.”
     - “Give me a quick wellness tip for today.”

3. **Simulate User Interaction**
   - Create a simple interface or command-line system where users can type requests in natural language.

4. **Process User Inputs**
   - The LLM interprets the input and generates appropriate responses such as reminders, suggestions, or task summaries.

5. **Adapt Responses Based on Feedback**
   - Collect user feedback or preferences.
   - Modify prompts to generate more personalized responses.

6. **Optional Memory Integration**
   - Store simple preferences like preferred work time or exercise reminders to simulate adaptive behavior.

---

## Expected Output

The application should simulate a **Personal Productivity Assistant** with the following features:

### 1. Daily Task Manager
- Accept tasks using natural language.  
- Example: “Remind me to call mom at 6 PM.”
- Organize tasks based on priority and deadlines.
- Provide daily summaries of completed and pending tasks.

### 2. Smart Scheduler
- Schedule events and reminders automatically.
- Detect overlapping appointments.
- Suggest available time slots.

### 3. Wellness Tips Generator
- Provide daily health and wellness suggestions.
- Example tips:
  - Drink sufficient water.
  - Take regular screen breaks.
  - Do light stretching or walking.

### 4. General Query Assistant
- Answer simple questions related to productivity, schedules, or daily planning.

---

## Example Output (Response by LLM)

**User Input:**  
Remind me to finish my project at 7 PM.

**Assistant Response:**  
Task added successfully. I will remind you to finish your project at 7 PM today.

---

**User Input:**  
Give me a wellness tip for today.

**Assistant Response:**  
Take a short 5–10 minute break every hour while working on a computer to reduce eye strain and improve productivity.

## Result
The lab exercise resulted in the creation of a prototype concept for a personal assistant powered by large language models. Students were able to:

- Understand how to tailor LLM prompts to real-life applications.
- Foster creativity by designing features suited to their personal or academic needs.
- Learn prompt engineering techniques for effective AI interaction.
- Experience the practical applications of generative AI in solving everyday productivity challenges.
